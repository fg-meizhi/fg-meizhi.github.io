PR 《学习之道》

Nov 21, 2017 | 读书

打卡，划小圈，逆境

有感触的几个点：

  1. 划小圈，缩小广度，增加强度/深度，把精力集中一点，不要太过分散，持续练习
  2. 利用逆境，发现自己其他的可能，持续练习
  3. 障碍不是障碍，是有待征服的挑战
  4. 从错误中学习，不断反思自己

弘丹的书评：《学习之道》|如何从初学者成长为高手？
