PR 学习分享

Nov 27, 2017 | 读书

大家好，我是梅智，非常激动有机会在这里和大家分享我在这一月的时间里学习PhotoReading的感受。

在10月份的时候，我的内心很焦虑，从职业的发展和个人的情况考虑，有越来越多的知识需要学习和消化，但是自己的能力有限，感到力不从心，迫切的希望能找到突破口，偶然的机会接触到了PhotoReading，就抱着试一试的心态来报名学习。

来上课的时候，我是带着很多担心的：

  1. 我对PhotoReading是怀疑的，感觉哪有那么神奇的方法能让做到那么好的效果
  2. 另一方面，因为学费还是挺贵的，只有三天的课程，很担心自己在这么短的时间内不能学会PhotoReading
  3. 同学自我介绍的时候，感觉有很多特别厉害的伙伴，自己算是个笨学生

上课的时候，内心的担心和紧张让我整个人的状态都很紧，很不好，第一天的课程结束后，脑袋痛到炸裂，回家什么都没有做就直接睡觉了，第二天就开始发烧39度，后面两天的课程也是在疼痛和发烧中度过，也没敢吃药，因为担心吃药后会嗜睡，不能集中精力听课。三天的课程，我一边跟着老师走，一边内耗着就过去了。课程结束后又整整发烧了五天，一直在医院输液。这个过程特别的痛苦，自己的情绪给身体带来了特别负面的反应。同时也在反省自己，我来上课的目的是什么，自己要怎样调整。

身体状态恢复以后，我们的训练营也开始了，我带着之前的担心开始了练习。按照上课学习的步骤，一步步的进行PhotoReading。练习一段时间以后，慢慢的进入了状态。进入状态是因为我做到了以下几点：

  1. 坚持，做了比完美更重要，不管自己的做的好不好，要坚持做下去才知道，才有机会发现自己做的好和不好的地方
    1. 每天早晨做PhotoReading的前4步
    2. 晚上激活画导图
    3. 打卡的记录发给若然老师，等待她做点评
  2. 不断思考，在阅读的过程中，对所看的书籍做了分类，快速阅读、记笔记、学习方法、个人成长、知识管理
    1. 明确了自己需要解决哪些问题
    2. 解决这些问题需要看什么样的书籍
    3. 集中看这一类书籍，不用太多，三四本的样子，然后练习主题阅读，对相同主题的问题总结
    4. 根据老师的反馈不断调整自己在练习过程做的不正确的部分，还有自己的旧习惯，比如：特别怕遗漏书中的内容，就整本书都看，速度慢，画的导图也又多又大，在若然老师的指导下，慢慢的改进
  3. 选择适合自己的工具，因为我是一个程序员的原因，我对电子设备很依赖，所以我从硬件和软件两个方面选择工具，和大家分享一下
    1. 硬件方面，我使用12寸iPad Pro 和 Apple Pencil，因为纸质的本子很多的颜色的彩笔对我来说携带不方便，而且不能同步，不能做到随时查阅
    2. 软件方面，我主要看软件的两个点，第一软件要易用，第二要能全平台同步（电脑、手机、pad），这样我可以随时编辑，随时查看，不会遗漏自己的想法
      1. 做笔记，画思维导图，我用Notability
      2. 分享，设备间共享文件用AirDrop，iCloud，用Hexo写独立博客记录自己的读书笔记
      3. 看电子书，用IPad上的多看阅读，PDF Expert，因为iPad屏幕够大，看书很方便
      4. Mac上的电子书管理用Calibre，可以为电子书分类和打标签

最后，说说自己的收获和改变

  1. 养成了读书的习惯，对读书不再抗拒，原来读书对我是一件大事要一大段完整的时间才可以开始，否则就干脆不做，结果就是很少能真正开始。现在每天需要一个小时左右就可以完成，对我来讲有了可操作性，也更容易坚持。
  2. 养成了反思、总结的习惯，每天晚上记录今天做的好的和不好的地方，反思，找改进的方法
  3. 让自己成长有希望，一步步的开始形成自己的知识体系
  4. 通过读书学习到的技能运用起来，来解决自己遇到的问题，把书中的知识转化为自己的技能，做到这一点是我特别开心的

感谢PhotoReading给我带来的这些改变，让我这个笨学生也能有这么多的收获，也是因为能看到这些实实在在的收获，缓解了不少我内心的焦虑，希望坚持下去能给我带来更多的改变，也希望在今后的日子里面能和大家共同成长，感谢大家的聆听，谢谢。
