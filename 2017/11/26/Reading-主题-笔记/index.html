PR 主题阅读 - 记笔记

Nov 26, 2017 | 读书

打卡，主题阅读第三次练习，明确了几个细节问题。

目的

为什么做笔记？笔记是用来干什么的？

什么时候使用笔记？

记笔记付出了时间、精力，给我带来了哪些回报？

内容

时间

使用手账记录

  1. 每天做了什么
  2. 分类：工作、健康、个人、幸福
  3. 每周目标
  4. 每年目标

分类不用太细

情绪

负面情绪： 遇到问题后，自己是怎么面对的，内心是什么情绪，什么原因会产生这样的情绪？

正面情绪： 发现积极的情绪，自己成功的原因，积累起来，建设内心的自信

人际关系

安排和朋友会面的计划并执行。

技能

专业知识的知识体系，记录知识点，知识点的关系。

正确姿势

以终为始

写给未来的自己，记笔记的时候想象未来的自己在看笔记的时候，最想要看的是什么，怎么才能直接找到重点。

重点

重点不要太多，最多写三个，如果太多，挑重中之重。重点太多，就没有了重点。

秩序

笔记要有序，混乱无义。

体系

建立笔记体系，正确、合理的归类。

便于记忆

思维导图、图形、符号结合文字，合适的排版。

索引

建立索引，方便查找。尤其是当笔记数量上升到一定数量级之后，想找的内容很容易被淹没到大量的未整理笔记中。

整理

分类，建立搜集箱、整理箱。

秩序整理，每天执行。

使用

建立知识体系

工具

要能随时记录、随时查阅，全平台可同步
