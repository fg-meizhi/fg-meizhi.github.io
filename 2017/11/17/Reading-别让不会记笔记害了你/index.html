PR 《别让不会记笔记害了你》

Nov 17, 2017 | 读书

打卡，《别让不会记笔记害了你》比较有收获的几个点：

  1. 借助建立自己的知识体系，不断积累
  2. 要不断整理所记的笔记
  3. 搜集箱，每天去处理，有价值的归类，没有价值的丢弃
  4. 更新换代，练习记笔记的方式，提高记笔记的能力
  5. 为行动服务
